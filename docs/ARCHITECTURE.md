## 5. 公式推导链

```
┌──────────────────────────────────────────────────────────────────────┐
│              公式推导链: EV → ΔG → tier                                │
└──────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────┐
  │  Step 1: 期望价值 (Expected Value)                           │
  │                                                             │
  │              EV = BV + Σ(Gene_i × Φ_i)                      │
  │                                                             │
  │  其中:                                                      │
  │    EV  = Expected Value        期望价值                      │
  │    BV  = Base Value            基础价值 (链上资产/信誉)       │
  │    Gene_i = 第i个基因的表达强度   ∈ [0, 1]                    │
  │    Φ_i  = 第i个基因的权重因子    由DAO治理参数决定             │
  │                                                             │
  │  示例:                                                      │
  │    BV = 100                                                 │
  │    Gene_1(探索) = 0.8, Φ_1 = 1.2                           │
  │    Gene_2(协作) = 0.6, Φ_2 = 1.5                           │
  │    Gene_3(进化) = 0.9, Φ_3 = 2.0                           │
  │    EV = 100 + (0.8×1.2) + (0.6×1.5) + (0.9×2.0)           │
  │    EV = 100 + 0.96 + 0.90 + 1.80 = 103.66                 │
  └──────────────────────────────┬──────────────────────────────┘
                                 │
                                 │ EV_t 和 EV_{t-1}
                                 ▼
  ┌─────────────────────────────────────────────────────────────┐
  │  Step 2: 成长值 (Growth Delta)                               │
  │                                                             │
  │              ΔG = (EV_t - EV_{t-1}) / EV_{t-1}              │
  │                                                             │
  │  其中:                                                      │
  │    ΔG  = Growth Delta          成长变化率                    │
  │    EV_t   = 当前周期的期望价值                                │
  │    EV_{t-1} = 上一周期的期望价值                              │
  │                                                             │
  │  示例:                                                      │
  │    EV_{t-1} = 103.66                                        │
  │    EV_t     = 112.03 (新基因表达提升)                         │
  │    ΔG = (112.03 - 103.66) / 103.66 = 0.0807 (8.07%)        │
  │                                                             │
  │  ΔG 含义:                                                   │
  │    ΔG > 0  → 正成长, 智能体在进化                             │
  │    ΔG = 0  → 持平, 维持当前状态                               │
  │    ΔG < 0  → 衰退, 需要策略调整                               │
  └──────────────────────────────┬──────────────────────────────┘
                                 │
                                 │ ΔG 值
                                 ▼
  ┌─────────────────────────────────────────────────────────────┐
  │  Step 3: 分级分类 (Tier Classification)                      │
  │                                                             │
  │  ┌──────────────┬────────────────┬────────────────────────┐ │
  │  │ Tier 等级     │ ΔG 阈值条件     │ 权限/能力              │ │
  │  ├──────────────┼────────────────┼────────────────────────┤ │
  │  │ 🥉 Bronze    │ ΔG < 2%        │ 基础操作,观察模式       │ │
  │  │ 🥈 Silver    │ 2% ≤ ΔG < 5%   │ 中等操作,参与治理       │ │
  │  │ 🥇 Gold      │ 5% ≤ ΔG < 10%  │ 高级操作,提案权         │ │
  │  │ 💎 Diamond   │ 10% ≤ ΔG < 20% │ 核心操作,审批权         │ │
  │  │ 👑 Champion  │ ΔG ≥ 20%       │ 全部权限,跨链调度       │ │
  │  └──────────────┴────────────────┴────────────────────────┘ │
  │                                                             │
  │  示例续:                                                     │
  │    ΔG = 8.07%  →  5% ≤ 8.07% < 10%  →  🥇 Gold            │
  │    获得: 提案权 + 高级机器人操作权限                            │
  └─────────────────────────────────────────────────────────────┘


  ═══════════════════════════════════════════════════════════════
  完整推导链一图流:
  ═══════════════════════════════════════════════════════════════

  ┌─────────┐     ┌─────────────────────┐     ┌─────────┐
  │ 基因NFT  │────►│  EV = BV + Σ(G×Φ)  │────►│ EV 值   │
  │ Gene     │     │  期望价值计算        │     │ 数值    │
  └─────────┘     └─────────────────────┘     └────┬────┘
                                                     │
                    ┌─────────────────────┐          │
                    │  ΔG = (EVt-EVt-1)   │◄─────────┘
                    │       / EVt-1       │
                    │  成长率计算           │
                    └────────┬────────────┘
                             │
                    ┌────────▼────────────┐
                    │  tier 分级判定       │
                    │                     │
                    │  <2%  → Bronze      │
                    │  <5%  → Silver      │
                    │  <10% → Gold        │
                    │  <20% → Diamond     │
                    │  ≥20% → Champion    │
                    └────────┬────────────┘
                             │
                    ┌────────▼────────────┐
                    │  权限/能力分配        │
                    │  执行对应层级指令     │
                    └─────────────────────┘
```
